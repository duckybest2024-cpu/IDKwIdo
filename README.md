## Deploy to Railway (2 steps)

1. **Connect this repo to Railway**
   - railway.app → New Project → Deploy from GitHub
   - Select this repo, branch: `claude/game-identification-xrltba`
   - Railway auto-detects Node.js and runs `node server.js`

2. **Done.** No database, no login, no env vars needed.

## Local dev

```bash
npm install
node server.js
# Open http://localhost:3000
```

## Optional: set a custom port
```
PORT=8080 node server.js
```
